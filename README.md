## Run Locally

Clone the project

```bash
  git clone https://github.com/Drakunal/Advanced-Analytics-2024.git
```

Go to the project directory

```bash
  cd Advanced-Analytics-2024
```

Opening in VS Code

```bash
  code .
```

Opening in Jupyter notebook

```bash
  jupyter notebook
```
